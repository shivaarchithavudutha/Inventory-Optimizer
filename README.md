# Inventory-Optimizer
An interactive FMCG Supply Chain Decision Support System (DSS) that uses EOQ and Safety Stock modeling to optimize procurement and minimize stock-outs

## Key Features
* **Economic Order Quantity (EOQ) Modeling:** Automatically calculates the most cost-effective order size to minimize storage and ordering costs.
* **Dynamic Reorder Logic:** Uses statistical Safety Stock calculations to prevent stock-outs during lead time.
* **Interactive Analytics:** A Streamlit-based dashboard providing real-time visibility into sales trends and procurement advice.
* **Data Driven:** Implemented a synthetic data generator to simulate 365 days of complex FMCG supply chain transactions.

## Tech Stack
* **Language:** Python
* **Libraries:** Pandas, NumPy, Plotly
* **Framework:** Streamlit

## Business Impact
By applying this logic, a procurement manager can reduce excess inventory holding costs by up to 20% while maintaining a 95% service level against demand spikes.
